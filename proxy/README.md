# FoodProof DK — Vision proxy

A tiny Cloudflare Worker that holds the Anthropic API key, enforces the
free-tier daily quota for vision OCR, and (optionally) checks Pro
entitlement via the RevenueCat REST API.

## Why

Mobile clients can't be trusted with the Anthropic API key — `EXPO_PUBLIC_*`
env vars are bundled into the JS and easily extracted from a published app.
This Worker is the only thing that knows the key. Free quota also lives here
because a client-side counter is bypassed by reinstalling the app.

## Endpoints

```
POST /vision
{
  "deviceId":      "stable per-install id, generated by the client",
  "imageBase64":   "...",
  "mediaType":     "image/jpeg",
  "rcAppUserId":   "optional — RevenueCat App User ID"
}
```

Returns the Anthropic Messages API response verbatim on success, or:

| Status | Meaning |
|---|---|
| 400 | Invalid JSON or missing fields |
| 402 | Free-tier quota exceeded (`{ error: "quota_exceeded", used, limit }`) |
| 405 | Method not allowed |
| 5xx | Anthropic upstream error (passthrough) |

## Deploy

```sh
cd proxy
npm i -g wrangler

# 1. KV namespace for daily quota
wrangler kv:namespace create QUOTA_KV
# Paste the printed id into wrangler.toml under kv_namespaces[].id

# 2. Secrets
wrangler secret put ANTHROPIC_API_KEY
wrangler secret put REVENUECAT_SECRET_KEY     # optional, only if you check Pro

# 3. Deploy
wrangler deploy
```

You'll get a URL like `https://foodproof-dk-proxy.<account>.workers.dev`.
Set this as `EXPO_PUBLIC_API_BASE_URL` in the app's `.env`.

## Local development

```sh
wrangler dev
```

Defaults to `http://localhost:8787`. The Expo app honours
`EXPO_PUBLIC_API_BASE_URL` if set; otherwise it falls back to a direct
Anthropic call using `EXPO_PUBLIC_ANTHROPIC_API_KEY` (dev-only path).

## Tightening before launch

- Set `ALLOWED_ORIGINS` if you ever embed the API in a web build.
- Add per-IP rate limiting (Cloudflare's WAF rules or a second KV counter).
- Consider HMAC-signing requests with a per-install secret rather than a
  bare deviceId to make quota harder to game across fresh installs.
